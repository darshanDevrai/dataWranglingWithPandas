Final thoughts¶

The data contains verious breeds of Dogs. But there some variables which I have not extracted properly from the text part of the tweet like gender so for gender of dogs we need some extra data or another method of gender extraction. The data generated by Neural network played an important role as it gives us the dogs breed which is very helpful for insights. The #WeRateDogs twitter data is very interesting and surprising.
